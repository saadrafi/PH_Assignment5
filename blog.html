What is DOM and what is the purpose of DOM?

The Document Object Model (DOM) is a programming interface.It represent tree like structure of webpage.Every node of the tree is a object which represent the elements,attibute,text etc.
The purpose of DOM is that by using the DOM, we can write programs that can respond to user interactions, such as clicking a button, and update the web page content accordingly. This enables developers to create web pages that provide a more engaging and interactive user experience.

How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML elements using DOM by using various DOM methods. Here are some of the most commonly used DOM methods for selecting elements:

  1. document.getElementById()
  2. document.getElementsByClassName()
  3. document.getElementsByTagName()
  4. document.querySelector()
  5. document.querySelectorAll()

What is event bubble?

Event bubble in DOM means if there is a event handler in an elemet and it has a parent elements the the event handler of child element will trigger first.Then the parent's event handler will be executed, followed by its parent element's event handler, and so on, until the event reaches the top level element just like a bubble in water.

What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and is executed inside the other function. The purpose of a callback function is to allow a function to call another function and then receive a result or perform an action based on that result.
I use Callback function in event handler to perform some action when the event is triggered.So that I can perform some action when the event is triggered and I can reuse the event handler for multiple elements.
